## Tech Stack

- Next.js  
- React  
- Framer Motion
- Lenis
- CSS Modules or Tailwind CSS

## Getting Started

```bash
# Clone the repository
git clone https://github.com/urUsername/Business-Website.git
cd koys-photos
```

```bash
# Install dependencies
npm install
```

```bash
# Start the development server
npm run dev
```

Then open `http://localhost:3000` in your browser.
